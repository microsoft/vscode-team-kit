# Changelog

## 1.0.0 — 2026-05-13

Initial release.

### Added

- **research skill** — orchestrates multi-source research by decomposing queries, fanning out parallel subagents for web and code search, iteratively refining findings, and synthesizing structured reports with citations and confidence levels.
- **researcher agent** — research subagent with web search, GitHub search, and code reading tools; dispatched by the research skill as its primary workhorse.
